Debit Card UI

Description:
Your challenge is to create a realistic, non-interactive credit card UI using only HTML and CSS. This project will test your command of advanced CSS, including positioning, gradients, and pseudo-elements, to replicate a detailed real-world design without relying on any image files for the core components.

Live Demo Preview

Functional Requirements:

  • The page must center a single credit card element.
  • The card must include correctly positioned text for a card number, cardholder name, and expiry date.
  • You must create a chip icon and a logo (e.g., Mastercard) *using only CSS*. No tags are allowed for the chip or logo.
  • Use specific fonts like 'Roboto Mono' and 'Poppins' for an authentic look and feel.

Styling Suggestions:

  • Use position: absolute to precisely place elements like the chip, logo, and text sections on the card.
  • Leverage pseudo-elements (::before and ::after) to construct complex shapes like the Mastercard logo without extra HTML.
  • Use a dark, professional background for the card with light-colored text for strong contrast and readability.
  • Pay attention to details like font families, letter spacing, and font sizes to match a real credit card.
