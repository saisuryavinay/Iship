Problem Statement

Title: Dual Panel Code Editor with Run Output Preview

Description:
Create a dual-panel interactive code editor page with the following features:

  • A left panel that contains a tab-based interface: Description, Output, and Editorial.
  • A right panel that contains three code editors (HTML, CSS, and JS), each in a toggleable format.
  • Two Run buttons below the editor: one for in-app preview and another to open a new tab with output.

Reference Animation:

Constraints:

  • Use <textarea> for the HTML, CSS, and JS editors.
  • Use JavaScript to dynamically render the output into an <iframe> or new window.
  • Maintain a tab layout for switching views (Description, Output, Editorial).
  • Buttons and editors should have styled hover states and clear visual hierarchy.
  • Responsive design is optional but recommended for bonus points.
