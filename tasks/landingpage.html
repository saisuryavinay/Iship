Interactive Watch Product Landing Page

Description:
Your challenge is to build a modern, interactive product landing page for a luxury watch. This project will test your HTML structure and CSS layout skills, along with your ability to use JavaScript to create a dynamic user experience.

Live Demo Preview

Functional Requirements:

  • Create a responsive product grid layout with an image gallery on the left and product details on the right.
  • The image gallery must contain one main product image and several smaller thumbnail images below it.
  • Implement JavaScript functionality so that clicking on a thumbnail image updates the src of the main image to display the clicked thumbnail's picture.
  • Include a quantity selector with "+" and "-" buttons. The HTML and CSS should be structured to allow for future JavaScript functionality.
  • Ensure the page includes a prominent "Add to Bag" button.

Styling Suggestions:

  • Use a clean and professional font, such as 'Inter' from Google Fonts.
  • The page layout must be responsive, stacking the gallery and details vertically on smaller screens.
  • Apply a visual indicator to the currently active thumbnail, such as a colored border, to show which image is currently selected.
  • Design a clear header containing a logo and navigation links.
