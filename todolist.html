Problem Statement

Title: Print "To-do List"

Description:
You are required to build a simple web-based To-Do List application using HTML, CSS, and JavaScript where users can type in tasks they want to remember, add them to a visible list, mark them as completed when done, and delete them if no longer needed; the interface should be clean and user-friendly, and the interactivity should happen without reloading the page.

Constraints:

  • The application must allow users to add tasks dynamically using an input field and a button.
  • Clicking on a task must visually mark it as completed (e.g., using a strikethrough or color change).
  • Each task should have a delete option to remove it from the list.
